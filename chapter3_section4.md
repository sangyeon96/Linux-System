# Section 4. 파일 시스템

### 4.1 파일 시스템이란?

파일 시스템이란 애플리케이션이 스토리지에 데이터를 읽고 쓸 때 이를 중개하는 커널 서비스 중 하나이다.

애플리케이션 또는 사용자가 애플리케이션을 매개로 특정 데이터를 읽어 들이는 지시를 내렸을 때 애플리케이션은 **시스템 콜**을 이용해 데이터를 불러온다. 시스템 콜을 이용해 스토리지에서 데이터를 읽어들이는 과정 중간에 **파일 시스템**이 들어간다.
